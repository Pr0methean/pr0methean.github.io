Class EntropyBlockingHelper

  • java.lang.Object
    • io.github.pr0methean.betterrandom.prng.EntropyBlockingHelper
  • All Implemented Interfaces:
    Serializable

    public class EntropyBlockingHelper
extends Object
implements Serializable
    Provides common functionality used by EntropyBlockingRandomWrapper and thread-locally by EntropyBlockingSplittableRandomAdapter. Internally coupled with BaseRandom because the use of this class is a substitute for mixin inheritance.

      • checkMaxOutputAtOnce

        public void checkMaxOutputAtOnce()
        Ensures that the attached PRNG can output 64 bits between reseedings, given that its current entropy is the maximum possible. Methods such as Random.nextLong() and Random.nextDouble() would become much slower and more complex if we didn't require this.
        Throws:
        IllegalArgumentException - if the PRNG cannot output 64 bits between reseedings
