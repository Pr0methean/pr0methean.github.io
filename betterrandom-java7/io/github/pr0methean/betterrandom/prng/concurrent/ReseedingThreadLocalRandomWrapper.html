        Wraps the given Supplier. Uses the given RandomSeederThread to reseed PRNGs, but not to initialize them unless the Supplier does so. This ThreadLocalRandomWrapper will be serializable if the Supplier is serializable.
        Parameters:
        initializer - a supplier that will be called to provide the initial BaseRandom for each thread.
        seedGenerator - The seed generation strategy whose RandomSeederThread will be used to reseed each thread's PRNG.
        Throws:
        SeedException
