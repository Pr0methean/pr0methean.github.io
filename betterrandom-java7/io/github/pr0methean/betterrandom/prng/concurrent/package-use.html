Uses of Package
io.github.pr0methean.betterrandom.prng.concurrent

  • Packages that use io.github.pr0methean.betterrandom.prng.concurrent 
    Package Description
    io.github.pr0methean.betterrandom.prng.concurrent
    Implementations of Random designed to be used concurrently from multiple threads, backed by instances of SplittableRandom or ThreadLocal<?>.
  • Classes in io.github.pr0methean.betterrandom.prng.concurrent used by io.github.pr0methean.betterrandom.prng.concurrent 
    Class and Description
    BaseSplittableRandomAdapter
    Abstract class for implementations of BaseRandom that wrap one or more SplittableRandom instances.
    DirectSplittableRandomAdapter
    Abstract subclass of BaseSplittableRandomAdapter where DirectSplittableRandomAdapter.setSeed(long) and BaseRandom.setSeed(byte[]) replace the SplittableRandom that's used in the context in which they are called.
    RandomWrapper
    Wraps any Random as a RepeatableRandom and ByteArrayReseedableRandom.
    ReseedingSplittableRandomAdapter
    Like SplittableRandomAdapter, but uses a RandomSeederThread to replace each thread's SplittableRandom with a reseeded one as frequently as possible, but not more frequently than it is being used.
    ReseedingThreadLocalRandomWrapper
    A ThreadLocalRandomWrapper that reseeds all its instances using the RandomSeederThread for its SeedGenerator.
    ThreadLocalRandomWrapper
    Wraps a ThreadLocal<BaseRandom> in order to provide concurrency that most implementations of BaseRandom can't implement naturally.
