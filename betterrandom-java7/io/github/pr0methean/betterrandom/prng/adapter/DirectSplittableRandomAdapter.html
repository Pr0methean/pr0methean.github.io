    public abstract class DirectSplittableRandomAdapter
extends BaseSplittableRandomAdapter
    Abstract subclass of BaseSplittableRandomAdapter where setSeed(long) and BaseRandom.setSeed(byte[]) replace the SplittableRandom that's used in the context in which they are called. See ReseedingSplittableRandomAdapter for an example of when it does not make sense to extend this class.
