    public class AesCounterRandom
extends CipherCounterRandom

    CipherCounterRandom using AES (Rijndael).

    Keys larger than 128 bits, and thus seeds larger than 256 bits, require unlimited strength cryptography policy files on closed-source JDKs.

    NOTE: Because instances of this class require 128-bit seeds, it is not possible to seed this RNG using the CipherCounterRandom.setSeed(long) method inherited from Random until the seed array has been set.
