Class SimpleRandomSeeder

  • java.lang.Object
    • io.github.pr0methean.betterrandom.util.Looper
      • io.github.pr0methean.betterrandom.seed.SimpleRandomSeeder
  • All Implemented Interfaces:
    Serializable
    Direct Known Subclasses:
    LegacyRandomSeeder

    public class SimpleRandomSeeder
extends Looper
    Thread that loops over ByteArrayReseedableRandom instances and reseeds them. No EntropyCountingRandom will be reseeded when it's already had more input than output.

    Chris Hennick
