    public enum DefaultSeedGenerator
extends Enum<DefaultSeedGenerator>

    Seed generator that is the default for the program where it is running. PRNGs that are serialized using one program's default and deserialized in another program will use the latter's default seed generator.

    The default implementation maintains multiple strategies for seed generation and will delegate to the best one available at any moment. It uses, in order of preference:

    1. DevRandomSeedGenerator with 128-byte buffer
    2. RandomDotOrgSeedGenerator.DELAYED_RETRY with 625-byte buffer
    3. SecureRandomSeedGenerator with no buffer
    Author:
    Daniel Dyer
