Interface SeedGenerator

  • All Superinterfaces:
    Serializable
    All Known Implementing Classes:

    public interface SeedGenerator
extends Serializable
    Strategy interface for seeding random number generators. Implementations that can have multiple equivalent instances should implement Object.equals(Object) and Object.hashCode() to support RandomSeederThread.
    Author:
    Daniel Dyer, Chris Hennick

      • isWorthTrying

        boolean isWorthTrying()
        Returns true if we cannot determine quickly (i.e. without I/O calls) that this SeedGenerator would throw a SeedException if generateSeed(int) or generateSeed(byte[]) were being called right now.
        Returns:
        true if this SeedGenerator will get as far as an I/O call or other slow operation in attempting to generate a seed immediately.
