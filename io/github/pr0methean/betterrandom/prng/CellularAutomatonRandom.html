    public class CellularAutomatonRandom
extends BaseRandom

    Java port of the cellular automaton pseudorandom number generator developed by Tony Pasqualoni.

    NOTE: Instances of this class do not use the seeding mechanism inherited from Random. Calls to the setSeed(long) method will have no effect. Instead the seed must be set by a constructor.

    Author:
    Tony Pasqualoni (original C version), Daniel Dyer (Java port)
