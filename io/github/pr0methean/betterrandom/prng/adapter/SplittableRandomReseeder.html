Class SplittableRandomReseeder

  • java.lang.Object
    • io.github.pr0methean.betterrandom.prng.adapter.SplittableRandomReseeder

  • public final class SplittableRandomReseeder
extends Object
    Utility class to re-seed a SplittableRandom instance using reflection and Unsafe.putLongVolatile(Object, long, long). Since SplittableRandom doesn't support reseeding through its public API, this is the only way a ReseedingSplittableRandomAdapter can avoid a ridiculous load on the garbage collector.
