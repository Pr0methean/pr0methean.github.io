    • entropyBits

      public long entropyBits()
      Description copied from interface: EntropyCountingRandom
      Returns an estimate of the current amount of entropy.
      Specified by:
      entropyBits in interface EntropyCountingRandom
      Returns:
      The current estimated amount of entropy. Every time the PRNG is reseeded, the entropy count is set to the new seed's length; and every time it is used, it is decreased by the number of random bits in the output rounded up. The amount of entropy can go below zero, giving an indication of how far the entropy has been stretched. This estimate is a lower bound if the seed is perfectly random and is not being reused.
